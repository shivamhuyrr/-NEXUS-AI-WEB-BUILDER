# Contributing to NEXUS AI WEBSITE BUILDER

Thank you for your interest in contributing to NEXUS AI! We welcome contributions from the community to help make this AI-powered website builder even better.

## How to Contribute

### 1. Reporting Bugs

- Use the GitHub Issues tab to report bugs.
- Provide a clear description of the issue and steps to reproduce it.

### 2. Suggesting Enhancements

- If you have an idea for a new feature, open an issue to discuss it.

### 3. Pull Requests

1. Fork the repository.
2. Create a new branch (`feat/your-feature` or `fix/your-fix`).
3. Make your changes and commit them with descriptive messages.
4. Push to your fork and submit a Pull Request.

## Development Setup

Refer to the [SETUP_GUIDE.md](SETUP_GUIDE.md) for detailed instructions on setting up your local environment.

## Code Style

- Follow the existing code structure.
- Use meaningful variable names.
- Ensure your code is well-documented.
